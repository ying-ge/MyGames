# 贪吃蛇游戏 🐍

一个经典的贪吃蛇游戏，使用纯HTML、CSS和JavaScript实现。

## 游戏特点

- 🎮 经典贪吃蛇玩法
- 🖼️ 使用精美的图片作为食物素材
- 🎯 三种难度选择（简单、中等、困难）
- ⚡ 动态加速机制
- 💾 本地最高分记录
- 📱 响应式设计，支持移动端

## 游戏说明

### 操作方式
- 使用**方向键**（↑↓←→）控制蛇的移动方向
- 按**空格键**暂停/继续游戏
- 点击按钮可以开始、暂停、重新开始游戏

### 游戏规则
1. 控制蛇吃食物（图片），每吃到一个食物得10分
2. 蛇每吃到一个食物，身体会增长一节
3. 每吃5个食物（50分），游戏速度会自动提升
4. 撞到墙壁或自己的身体游戏结束
5. 最高分会自动保存到本地

### 难度设置
- **入门级**：速度最慢，非常适合第一次玩的新手
- **简单**：速度较慢，适合熟悉游戏的玩家
- **中等**：速度适中，平衡挑战性
- **困难**：速度较快，考验反应能力

## 技术实现

- **HTML5 Canvas**：用于游戏画面渲染
- **纯JavaScript**：实现游戏逻辑
- **CSS3**：现代化UI设计和动画效果
- **LocalStorage**：保存最高分记录

## 如何运行

1. 直接打开 `index.html` 文件即可开始游戏
2. 或者使用本地服务器运行：
   ```bash
   # 使用Python
   python -m http.server 8000
   
   # 使用Node.js
   npx serve
   ```
3. 在浏览器中访问游戏
