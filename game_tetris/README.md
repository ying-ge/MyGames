# 俄罗斯方块

经典俄罗斯方块游戏，采用古老的黑白液晶显示器风格。

## 游戏特色

- **复古风格**：黑白液晶显示器风格，绿色边框和扫描线效果
- **等级系统**：0-9级循环，每消除10行升一级，9级后回到0级
- **经典玩法**：7种经典俄罗斯方块形状（I、O、T、S、Z、J、L）
- **本地记录**：自动保存最高分

## 操作说明

- **← →** : 左右移动方块
- **↑** : 旋转方块
- **↓** : 快速下落
- **空格** : 暂停/继续游戏

## 游戏规则

1. 方块从顶部下落，使用方向键控制移动和旋转
2. 当一行填满时，该行会被消除，获得分数
3. 每消除10行，等级提升1级（0-9循环）
4. 等级越高，方块下落速度越快
5. 当方块堆到顶部无法放置新方块时，游戏结束

## 分数系统

- 消除1行：40 × (等级+1) 分
- 消除2行：100 × (等级+1) 分
- 消除3行：300 × (等级+1) 分
- 消除4行：1200 × (等级+1) 分
- 快速下落：每格1分

## 技术说明

- 纯前端实现，无需服务器
- 使用 Canvas 渲染
- 像素化渲染效果，还原复古显示器风格
- 响应式设计，支持移动端
